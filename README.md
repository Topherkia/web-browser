# Web Browser with Rating System

A desktop web browser application with integrated webpage rating and commenting system.

## Features
- Multi-search engine support (Google, Bing, DuckDuckGo, etc.)
- User authentication system
- Rate and comment on any webpage
- MariaDB/MySQL database integration
- Desktop application using Electron

## Quick Start
1. Run `scripts\install_dependencies.bat`
2. Run `database\init_database.bat`
3. Run `start.bat`
