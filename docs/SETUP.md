# Web Browser Rating System - Setup Guide

## Prerequisites

### 1. Software Requirements
- **Node.js** (v14 or higher) - [Download](https://nodejs.org/)
- **MariaDB/MySQL** (v10.5 or higher) - [Download MariaDB](https://mariadb.org/download/)
- **HeidiSQL** (for database management) - [Download](https://www.heidisql.com/)
- **Git** (for version control) - [Download](https://git-scm.com/)

### 2. Hardware Requirements
- **RAM**: Minimum 4GB (8GB recommended)
- **Disk Space**: Minimum 500MB
- **Processor**: Dual-core 2GHz or higher
